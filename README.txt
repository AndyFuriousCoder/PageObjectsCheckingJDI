
В данном проекте создаются и проверяются объекты сайта: https://jdi-framework.github.io/tests

В проекте используется Framework JDI для создания объектов страницы и работы с ними.

В проекте реализованы:

1) Проверка формы авторизации на сайте с применением Data provider на домашней странице;
2) проверка CheckBox'ов на странице Different Elements.
